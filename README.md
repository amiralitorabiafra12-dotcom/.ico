# .ico

## این برنامه برای تغییر فرمت تصاویر .png و .jpg به .ico ساخته شده و دارای قابلیت تغییر فرمت سریع و فقط با گرفتن آدرس ورودی و خروجی می تونه فایلتون رو درست کنه

## سیستم مورد نیاز:
- ویندوز 7 32 بیتی یا بالا تر
- رم 2 یا بالا تر
- برنامه دارای صفحه اختصاصی نیست و برای باز کردن آن حتما باید خط فرمان (command propt or cmd) داشته باشید

## راهنما :
1.فایل را دانلود کنید
2.فایل دانلودی را باز کنید
3. مراحل را طی کنید
4.آدرس فایلی که میخواهید به برنامه بدهید را بنویسید .برای مثال : C:\Users\RAYANEH\Untitled.png
5.آدرسی را که میخواهید فایل پس از تبدیل تصویر به آنجا برود را بنویسید .برای مثال : C:\Users\RAYANEH\Untitled.ico
